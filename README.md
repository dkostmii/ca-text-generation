# ca-text-generation

This project uses 1D CA (Cellular Automata) to generate random text from vowels and consonats alphabet.

Example output:

```text
Pipisqit aatqiwisehyt

        Bebceybe kykyqpyki pipiseutit azaz kafoiai bebyebe yituoifykoif ywoukif, cyraafmyh ititiqy kiz kizaizbiz eyebebye!
        Ituif yzvaf kapanlizyziz kif iaiifryz.
        Eybebyebeb luiqebceye beceb ubeybe obeb yaizbiziv; yazlazo tipysdykikyk bebcebeb; pateateyat: yteatkatahlat - kiziv ebe yahehah yaiz yizaiza becebyebe veb, aty atatakey akyaplizy kizice: `yzyz qo ytat yat`.
        Beceb ykykyqiykqyk azyziz aeatkateyat pykyc - becebde ebebceybeceb atkat yznataw, it ykiu - bebybe itit kafacoyz dudi kat yzgif: yzbiz ytatak yifa yzuap - yaiz yeiteyetit itatuik yykqyk obucebde?
        Yiykqyk: hiziq uoleyebeceb, itkatetate iateyat uqeyeyebeybe ebebce.
        Zyfynikyk kikekikijrat atatetateytat akekakan fykym yzizaiz; yazazoaz kitaeitqaz fat akuikyk, yk ube ebebybeb yziz zat kiz beybe fytoiatit: pipeuipeit tipoiy yzfikuyk cyfaytyzam beybe ebeceb yziz; fateyatkat ytueityepip azaze pazaxlaz zatavoizeat itbizid; tate zapopapo yhacap yzizilpato qevuobeybe yzezazal veq yipepip, ytiteyiteit paficykad atye izuipdykym becebybebye oleveb zatahlat!
        Kykyqiykiui yzaoaz yazlaz zazar izozyz oqey; iateytatey iyfnikouou iaiza qubebybe yazizaziz: kizaziz.
        It kiz yeitrip, zat yteat ytetataketat, ak kah iiz - ebebybeby hiwacsifyc dudfududfu aotitivqaz pat kateatkat, iufyfyliiko!
        Zafys iteatkateat fafoyzvaf iyfifyf yeyeytiteyit aatlazaheatat, azaz bebceybeby; aeacywim vubceyebeb iklyfygnik kiz kipe: aeatkat tizic lu kazoa yzgificryz beceb yzipakyli itat - yzizya patetatak fatajeyetat oualeq!
        Zyf; fateyat katah pit ytbizeziz yzhapanliz ytkatahlat, katawlizihpat yzozyzozyz.
        Beye katah yateytatak azrazax; loxelavoyu!
        Tat ytit itkat kafo iizea yateyeat yz ey; tike kat yh pizicrytym kite ahahehahe?
        Geibebce yz yzlatoeae iteytitiq yao atate.
        Titeite azmyf zafo aflywin, izyap: aheaheahnah iizeyta iw yoiz kapoizhapan, yatkat kipeueitrip kat kazoziz yip ytkatah bebcebeb izyf, beybebce katah ykikyk ytity iapoyziz izbiz: yeyatkatahe ebeceby fizivqizima ebecebde yzazoaz - yzbiz hykym yitufif?
        Azazarlazah - 'yzuaplizyziz fitofyfipqif becebybeceb kifayzgifafif kiteytit ikokikinoyf yatkat vevgeiveuvo'; fazoazazal.
        Ay, yeate zitimritey yzafanukyk kit yey: (yzgifafif zatahlat tatakeat yzafiuoi beybe oulevo).
        Eyebe yazazoao bebceybe du ayrouimeap ejejujejke kapanliz: yhqak bebybebceybe: yeitetite yzaz tateat yiykiuiyk vev ytripiwqit kizaizao; lobebybe fapefap yteit?
        Itita bebybeb - ye kifafif ebe azizyzylozyz ebey zitim.
        Ithakymiukyk hykikykyq ywah pazaxlazah bebybeb - bebyey levtedug iizqiza atbiz ytqit ovebeb itaeytbizid bebybeceb!
        Zyzozyz yhcip yhawyvnak zaf beb zazo.
        At ykykyqiykqyk - yipritimyeu?
        Yhiak ytateatkat fyzozyzoai ykizaxofit ytqitim izazezaz kikeiik kazozaz: ytat ietit yaz yatitae!
        Ayacsypuac azeit ytiawzytiaw zafoiayzvaf ytkateyeyat; `yatoeaiznat ie ytetit yzizea yokyf`, yfawinuhyhyt yteatetat, ititetitiq aknyh - yziz zazaroa izazeit: kat yahe izbiza leq ytqitim katizrim tatetatakeat lubebcebeb: ytytatyt?
        Atataloea beybebyey kitaaztitiv - ytateatkat kaf ytkatah iatkat yte duidufud hyriqhafetap pitoiufif - kipeutit ytkat yaoizaziz fataqotat kityc vevybey!
        Itqitetit, gev aktawiheuyc kizivqiz ebeybeybeb qebyebeb, fazo ykykyqpykikyk kaf aatlaz yipepipeit ytateatkate.
        Becebybeybe pitimeitetit yzazoaz yzaz ea - yzokifychyf aapoyzizily bey yaiz yzmyfil zaz yeitqiteytit!
        Ap yhehahe beybe: pikihrat azazozaz yuyacyuacbyp pitoiuoi; tiz yazazaro lesynopon itripiwqit yk papahmizyziz - yzfik fato zityx yat luiqebce kipiw ytataklat itbize!
        Azozazoaz; katahlatey kazaxlazoa yaoziza uduidufudi ey ytkatetatak ytqiteit - ebeb: (yhacaryuirym azhap ytqit yatkatahe) - yzbiziv ytyt tat aeye afaw fatajlat fit atkatetate; ytripiwqit yizeytbizid tip!
        Wahiak ytit azaz katahlatah kat ieaizqyt yizqiz: "eyebeb yzazezaz yt yzo kiz kiz": pip yzo; fifafif ebebce fapan yeahnah lobeby; aatkat gegcemeq iuykiykikyk?
        Tatak yzlat ytatak; yhiakeyhi iahu fitaazezaz ieitqitim ytytyqatyt, iklyfyg eybeceb atotytaea kykikykyq bebybe aoazozaz itkatahlat ikik: pipepipis iitsif azaf tat patah tifil - yt veq yay!
```

# How to run

Open the project in Visual Studio 2022 and run it with default configuration.
